## 역할별 책임 기준

- 모든 역할은 작업 결과물을 Google Drive에 올리고, 관련 Issue에 링크를 남깁니다.
- 담당자는 작업 진행 상황을 Issue 댓글로 공유합니다.
- 검수자는 과제 요구사항 기준으로 수정 필요 여부를 판단합니다.
- PM은 최종 링크가 README와 final-submission-links.md에 반영되었는지 확인합니다.
