# Figma 작업 가이드

## Figma 프로젝트 목적

Figma 프로젝트는 AI로 생성한 UI 이미지를 화면 흐름에 맞게 배치하고, 사용자가 어떤 순서로 화면을 이동하는지 보여주기 위한 공간입니다.

## 화면 배치 원칙

- 메인, 목록, 상세 화면을 한눈에 비교할 수 있게 좌에서 우로 배치합니다.
- 화면 크기와 비율은 가능한 한 통일합니다.
- 최종 이미지와 실험 이미지를 섞지 않습니다.
- 최종 제출에 사용할 화면에는 이름과 버전을 명확히 표시합니다.

## 화면 이름 규칙

```text
ProjectA_화면역할_v번호
예: ProjectA_Main_v01
```

권장 화면 역할:

- Main
- List
- Detail
- Result
- Login
- Setting

## Hotspot 또는 화면 전환 표시 방법

1. Figma의 Prototype 탭을 엽니다.
2. 클릭 가능한 버튼이나 카드에 연결점을 만듭니다.
3. 다음 화면 프레임으로 연결합니다.
4. 전환 방식은 Instant 또는 Smart Animate 중 팀에서 통일합니다.
5. Preview에서 실제 흐름이 자연스러운지 확인합니다.

## 최종 링크 공유 전 확인사항

- 링크 접근 권한이 `링크가 있는 사용자 보기 가능` 또는 제출 기준에 맞게 설정되어 있는지 확인합니다.
- Hotspot이 끊긴 곳이 없는지 확인합니다.
- 최종 화면만 남기거나, 최종 화면 영역을 명확히 표시합니다.
